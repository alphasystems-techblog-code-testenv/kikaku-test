# kikaku-test

このプロジェクトは、技術ブログの GitHub 公開用プロジェクトのテスト用リポジトリです。
GitHub への公開プロセスなどを目的としています。

## プロジェクト構成

- `hello.py`: シンプルな Python スクリプト。
- `test_hello.py`: `hello.py` の動作を確認するテストスクリプト。
- `requirements.txt`: 依存関係ファイル（現在は空）。

## 実行方法

### スクリプトの実行

```bash
python3 hello.py
```

引数を渡すと、出力先の名前を変更できます。

```bash
python3 hello.py world
```

### テストの実行

```bash
python3 test_hello.py
```
